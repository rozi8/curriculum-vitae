🚀 Tutorial: Deploy Nginx & Redis secara Otomatis dengan Terraform dan Docker

Mini proyek pembelajaran DevOps menggunakan pendekatan Infrastructure as Code

Halo, DevOps enthusiast! 👋
Pada artikel kali ini, saya akan membagikan sebuah mini-proyek pembelajaran yang saya kerjakan untuk mendalami automasi infrastruktur menggunakan Terraform dengan Docker provider. Tujuannya adalah melakukan deployment container Nginx dan Redis secara otomatis, dengan pendekatan yang modular, bersih, dan mudah untuk dikembangkan.

Proyek ini cocok untuk kamu yang sedang belajar Infrastructure as Code (IaC) dan ingin memahami bagaimana mengelola layanan kontainer dengan pendekatan yang lebih terstruktur.

🔧 Apa yang Akan Kamu Dapatkan?

  • ✅ Deployment otomatis untuk container Nginx dan Redis menggunakan Terraform.
  • 🧱 Struktur modular, sehingga setiap layanan dikelola dalam modul terpisah.
  • ⚙️ Variabel input terpisah menggunakan terraform.tfvars, agar mudah disesuaikan.
  • 📤 Menampilkan output penting setelah provisioning selesai.
  • 🗂️ Struktur direktori rapi dan mudah dipahami, memudahkan kolaborasi maupun pemeliharaan.

📁 Struktur Direktori

.
├── main.tf
├── outputs.tf
├── terraform.tfvars
├── terraform.tfstate
├── terraform.tfstate.backup
├── variable.tf
├── README.md
└── modules/
    └── redis/
        ├── main.tf
        ├── output.tf
        ├── terraform.tfvars
        └── variables.tf

Setiap layanan (seperti Redis) berada dalam direktori modules/ agar mudah digunakan kembali di proyek lain.

🚀 Langkah-langkah Penggunaan

  1. Clone repository 
    git clone https://github.com/rozi8/infra-nginx-redis-modular
cd infra-nginx-redis-modular
  2. Duplikat file variabel contoh 
    cp terraform.tfvars.example terraform.tfvars

    Edit terraform.tfvars sesuai kebutuhan kamu, seperti port dan konfigurasi lainnya.

  3. Jalankan perintah Terraform 
    terraform init
terraform plan
terraform apply
terraform output
  4. Akses Nginx melalui browser: 
    http://localhost:8087

📌 Catatan Tambahan

  • Pastikan Docker sudah terinstall dan berjalan di sistem kamu.
  • Project ini menggunakan Docker provider dari Terraform, jadi kamu tidak memerlukan Docker Compose.
  • Kamu bisa menambahkan modul lain seperti PostgreSQL, MongoDB, atau aplikasi lain dengan struktur modular serupa.
