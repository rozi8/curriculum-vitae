🖧 Topologi Infrastruktur Jaringan

Penerapan VLAN, DHCP, OSPF, dan Hotspot menggunakan MikroTik Router

📌 Ringkasan Konfigurasi

  • ✅ DHCP Server: Aktif di MikroTik Server
  • ✅ OSPF Routing: Terkonfigurasi antar MikroTik
  • ✅ Hotspot: Berjalan di MikroTik Hotspot (dengan Firefox sebagai client)
  • ✅ VLAN: Terbagi ke dalam VLAN 10 & VLAN 20
  • ✅ Firewall: Tersedia di MikroTik Hotspot & MikroTik Lab
  • 🔧 Fungsi routing dinamis dan statis aktif pada beberapa router

🌐 Zona Jaringan

🟦 Zona Biru - Lab

  • MikroTik Lab sebagai gateway
  • Switch dengan 3 client (Lab-1, Lab-2, Lab-3)
  • Terhubung ke MikroTik Junction → MikroTik Server

🟩 Zona Hijau - Server

  • MikroTik Server sebagai DHCP utama
  • OSPF aktif untuk routing ke semua MikroTik
  • Hotspot tersedia pada MikroTik Hotspot

🟨 Zona Kuning - VLAN & Guru

  • MikroTik Ruangan terhubung ke VLAN 10 & VLAN 20
  • Switch ke PC Guru (VLAN 10)
  • Switch ke Switch 2 (VLAN 20)
  • Trunk port antar Switch

🔧 Fitur Implementasi

  • 🔁 Dynamic Routing dengan OSPF
  • 🛡️ Firewall di MikroTik Hotspot & Lab
  • 📶 Hotspot aktif dengan login captive portal
  • 🌐 VLAN tagging dan trunk antar switch
  • 📦 DHCP Server dan Client antar subnet
