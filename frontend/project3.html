🚢 Desain Kapal 3D Multisoftware

Proyek desain kapal menggunakan Maxsurf, AutoCAD, dan SketchUp

📌 Ringkasan Proyek

  • Dimensi kapal: Panjang 820 mm, Lebar 250 mm, Tinggi 274 mm
  • Dirancang secara bertahap dari Maxsurf → AutoCAD → SketchUp
  • Detail meliputi dek, jendela, helipad, dan propeller
  • Model siap untuk 3D printing dan rendering visual

🔧 Tools & Software yang Digunakan

  • Maxsurf: untuk perancangan awal hull kapal
  • AutoCAD: untuk penggambaran struktur detail 2D atau lines plan
  • SketchUp: untuk pembuatan model 3D dan simulasi bentuk akhir

📝 Catatan Desain

  • Dirancang untuk keperluan pembelajaran dan simulasi desain
  • Skala akurat untuk proporsi realistis miniatur kapal
  • Visualisasi 3D mendukung perspektif tampak atas, samping, dan isometrik
  • Format SketchUp bisa diekspor ke STL, DWG, dan lainnya
